API Project: URL Shortener Microservice

    User Stories :

  1. POST a URL to https://wiry-ski.glitch.me/api/shorturl/new and receive a shortened URL in JSON response.
    Example : {"original_url":"www.google.com","short_url":1}
  2. If an invalid URL is passed, that doesn't follow the http(s)://www.example.com(/more/routes) format, or does not point to a valid site, the JSON response will contain an error.
    Example: {"error":"invalid URL"}
  3. When the shortened URL is visited, it will redirect to the original link.

Short URL Creation

Example: POST https://wiry-ski.glitch.me/api/shorturl/new/https://www.google.com

Visiting the Shortened URL Example:

https://wiry-ski.glitch.me/api/shorturl/3

Will Redirect to:

https://www.google.com
